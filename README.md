# Shaurya Sharma — Portfolio

Personal portfolio showcasing my projects, skills, pricing, and selected work across AI, web, and product builds.

**Live site:** [portfolio-shaurya28.vercel.app](https://portfolio-shaurya28.vercel.app/)

## Highlights
- Clean, dark, responsive design
- Featured projects with image previews
- Skills, pricing, and contact sections
- Mobile-friendly across modern browsers

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS

## Run Locally
```bash
npm install
npm run dev
```
